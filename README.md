# üì± JTI One To One

**JTI One To One** is a tablet-based application built for Brand Ambassadors (BAs) to efficiently manage field campaign activities.

---

## üöÄ Purpose

The app provides BAs with a secure login (OTP or Face Recognition) and a personalized dashboard showing personal details, employment history, and performance. BAs can manage assigned campaigns, complete daily checklists, interact with clients and outlets, conduct surveys, present games, and access training materials ‚Äî all in one place.
